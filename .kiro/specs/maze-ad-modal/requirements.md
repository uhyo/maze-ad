# Requirements Document

## Project Description (Input)
A Vite-based front-end application. Should use React. The page contains an article. Contents are not important. When the user scrolls down, an application invokes and shows a modal that imitates an advertisement that covers the page that user needs to press the close button to continue. The contents of that "ad" should look like a casual maze game.
However, when the user tries to press the close button, it escapes into the maze.
Then the game starts. User needs to use keyboard arrow buttons or on-screen arrow button (for use by modile users) to explore the maze and reach the goal (close button). Then the application closes and user can continue reading.
